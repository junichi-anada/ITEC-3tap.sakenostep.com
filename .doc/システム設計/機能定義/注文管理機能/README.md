# 注文管理機能

## 概要
顧客からの注文を管理するための機能群です。顧客による注文とオペレータによる注文管理の2つの側面があります。

## 機能一覧
1. 顧客向け注文機能
   - 注文リストへの商品追加
   - 注文リスト内の商品数量変更
   - 注文リストからの商品削除
   - 注文の確定
   - 配送先情報の指定

2. オペレータ向け注文機能
   - 顧客指定での注文登録
   - 複数商品の一括登録
   - 配送先情報の登録・変更
   - 注文ステータスの管理
   - 注文メモの登録

3. 注文一覧機能
   - 注文履歴の表示
   - 検索・フィルタリング
   - ソート機能
   - ステータス別の表示

4. 注文詳細機能
   - 注文の詳細情報表示
   - 配送状況の確認
   - 関連ドキュメントの表示
   - 注文履歴の参照

5. 注文CSV出力機能
   - 注文データのCSV出力
   - 出力条件の指定
   - フォーマット設定
   - 一括ダウンロード
