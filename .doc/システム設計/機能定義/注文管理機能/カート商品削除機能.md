# カート商品削除機能設計書

## **1. 機能概要**
- **目的**: カートに追加済みの商品を削除する機能
- **対象者**: システムにログインした顧客
- **方式**: データベースとローカルストレージを併用したハイブリッド方式

## **2. 要件一覧**
### **必須要件**
- 単一商品をカートから削除できる
- 複数商品を一括でカートから削除できる
- カートデータをデータベースで管理
- カートデータをローカルストレージにキャッシュ

### **拡張要件**
- 削除時の合計金額の自動再計算
- 削除の取り消し機能（一定時間内）
- オフライン時の一時的な削除操作対応

## **3. 入出力仕様**
### **入力**
#### 単一商品削除
```json
{
    "item_id": 1
}
```

#### 複数商品一括削除
```json
{
    "item_ids": [1, 2, 3]
}
```

| 項目           | 型          | 必須 | 説明                          |
|---------------|-------------|------|------------------------------|
| `item_id`     | 整数        | ※1  | 商品ID（単一削除時）          |
| `item_ids`    | 配列        | ※1  | 商品IDリスト（一括削除時）     |

※1: `item_id`または`item_ids`のいずれかが必須

### **出力**
#### 削除時（単一/複数共通）
```json
{
    "status": "success",
    "message": "商品を削除しました",
    "data": {
        "cart_id": 1,
        "deleted_items": [1, 2],
        "remaining_total": 4500,
        "cache_timestamp": "2024-03-01T10:00:00Z"
    }
}
```

## **4. 処理フロー**
1. 商品削除処理
   - リクエストパラメータの形式チェック
   - 単一削除か一括削除かの判定
   - 各商品に対して:
     - 対象商品の存在チェック
   - トランザクション開始
   - 注文明細の一括論理削除
   - 合計金額の再計算
   - トランザクション確定
   - キャッシュの更新

## **5. API仕様**
### エンドポイント
- **カートからの商品削除**:
  - 単一商品削除:
    - URL: `/api/cart/items/{item_id}`
    - メソッド: DELETE
  - 複数商品一括削除:
    - URL: `/api/cart/items`
    - メソッド: DELETE
    - パラメータ: `item_ids[]`
  - Content-Type: application/json

### リクエストヘッダー
```
Authorization: Bearer {token}
X-CSRF-TOKEN: {csrf_token}
```

## **6. データ設計**
### データベース設計
| カラム名      | 型      | 説明                    |
|--------------|---------|------------------------|
| id           | INT     | 注文ID                 |
| user_id      | INT     | ユーザーID             |
| status       | ENUM    | 注文状態（CART/ORDERED）|
| total_amount | DECIMAL | 合計金額               |
| created_at   | DATETIME| 作成日時               |
| updated_at   | DATETIME| 更新日時               |

### ローカルストレージ設計
```json
{
    "cart": {
        "id": 1,
        "items": [
            {
                "item_id": 1,
                "name": "商品A",
                "price": 1000,
                "quantity": 2
            }
        ],
        "total_amount": 2000,
        "timestamp": "2024-03-01T10:00:00Z",
        "expires_at": "2024-03-01T11:00:00Z"
    }
}
```

## **7. バリデーション仕様**
- `item_id`: 必須（単一削除時）、カートに存在する商品ID
- `item_ids`: 必須（一括削除時）、カートに存在する商品IDの配列
- 削除対象の商品が全てカートに存在することを確認

## **8. エラーハンドリング**
### バリデーションエラー
```json
{
    "status": "error",
    "message": "商品の削除に失敗しました",
    "errors": {
        "item_ids": ["指定された商品の一部がカートに存在しません"],
        "item_ids.1": ["カートに存在しない商品です"]
    }
}
```

### システムエラー
```json
{
    "status": "error",
    "message": "システムエラーが発生しました",
    "errors": {
        "system": ["削除処理に失敗しました。しばらく経ってから再度お試しください。"]
    }
}
```

## **9. セキュリティ対策**
- 認証済みユーザーのみアクセス可能
- CSRFトークンによる保護
- 入力値のサニタイズ
- ユーザーごとのカート情報の分離
- キャッシュデータの暗号化
- キャッシュの有効期限設定

## **10. 特記事項**
### 削除の取り消し機能
- 削除後30分以内であれば取り消し可能
- 取り消し時は削除前の数量で復元
- 在庫状況により復元できない場合がある
