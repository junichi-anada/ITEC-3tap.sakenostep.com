# 3.2 データベース構成

## 認証管理テーブル群

### authenticates
- 認証情報を管理
- 主要カラム
  - auth_code: 認証コード（UK）
  - login_code: ログインコード
  - password: ハッシュ化パスワード
  - expires_at: 有効期限

### password_histories
- パスワード履歴を管理
- 主要カラム
  - authenticate_id: 認証ID（FK）
  - password: 過去のパスワード
  - created_at: 作成日時

### auth_providers
- 認証プロバイダ情報を管理
- 主要カラム
  - provider_code: プロバイダコード（UK）
  - name: プロバイダ名
  - api_key: APIキー

## 商品管理テーブル群

### items
- 商品情報を管理
- 主要カラム
  - item_code: 商品コード（UK）
  - name: 商品名
  - price: 価格
  - category_id: カテゴリID（FK）

### item_categories
- 商品カテゴリを管理
- 主要カラム
  - category_code: カテゴリコード（UK）
  - name: カテゴリ名
  - priority: 表示順序

### item_units
- 商品単位を管理
- 主要カラム
  - unit_code: 単位コード（UK）
  - name: 単位名

## 注文管理テーブル群

### orders
- 注文基本情報を管理
- 主要カラム
  - order_code: 注文コード（UK）
  - user_id: ユーザーID（FK）
  - total_price: 合計金額
  - ordered_at: 注文日時

### order_details
- 注文明細を管理
- 主要カラム
  - order_id: 注文ID（FK）
  - item_id: 商品ID（FK）
  - quantity: 数量
  - price: 価格
