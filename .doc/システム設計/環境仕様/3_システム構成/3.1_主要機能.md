# 3.1 主要機能仕様

## 認証管理 [優先度: 高]

### 顧客認証
- 認証方式
  - お客様コード + 電話番号による認証
  - JWTトークンの発行
- セキュリティ
  - アカウントロック（5回失敗で30分ロック）
  - パスワード有効期限（90日）

### オペレータ認証
- 認証方式
  - ID + パスワードによる認証
  - 2要素認証（将来対応）
- 権限管理
  - 管理者権限
  - 一般オペレータ権限
  - 閲覧専用権限

## 商品管理 [優先度: 高]

### 商品検索
- 検索条件
  - 商品名（部分一致）
  - カテゴリ
  - 価格帯
- 表示機能
  - 無限スクロール
  - ソート機能（価格順、登録日順）

### 商品登録/更新/削除
- 登録項目
  - 基本情報（商品名、価格、説明）
  - カテゴリ情報
  - 公開設定
- バリデーション
  - 必須項目チェック
  - 型チェック
  - 重複チェック

### おすすめ商品管理
- 設定機能
  - おすすめフラグ
  - 表示順序
  - 公開期間

## 注文管理 [優先度: 高]

### 注文登録/更新
- 登録機能
  - 商品選択
  - 数量指定
  - 配送先指定
- 更新機能
  - ステータス更新
  - キャンセル処理

### CSV出力機能
- 出力項目
  - 注文基本情報
  - 明細情報
  - 集計情報
- フォーマット
  - Shift-JIS
  - カンマ区切り
